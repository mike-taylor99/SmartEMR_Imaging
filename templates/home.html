Medical Imaging Web-Service

Upload Patient Images

Upload Patient Images to new or existing patient profiles.

Find Patient Images

Input a Patient ID (PID) and have all corresponding images & information returned. Will return a 404 message if no records are found.

Find Images by Tags and Descriptors

Input tags and descriptors in a comma-separated format (e.g., xrays, cavity, teeth) and have all images with those tags and descriptors returned.

Patient ID and Tags/Descriptors Querying

Input fields take in Patient ID (PID) or tags/descriptors (not exclusive). Endpoint will accept inputs with both, one, or none of the fields filled and return relevant images. This endpoint will return an array of the image file names to be used by the front-end interface for display purposes.

Natural Language Querying

Takes a natural language query as input and returns all relevant images which may be related to the query

Medical Image Classification

Accepts a medical image for input and returns an array of all descriptors which the deep-learning model percieves the image to be.
